🖨 Before You Print

Follow these steps in Chrome's print dialog for the cleanest output:

  • 1 Set Layout → Portrait
  • 2 Click "More settings"
  • 3 Uncheck "Headers and footers"
  • 4 Set Margins → None or Minimum
  • 5 Click Print
